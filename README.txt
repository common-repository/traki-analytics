Link up your WordPress website with Traki

== Description ==

Website analytics you'll actually understand.

A new way to understand your website, your visitors, the actions they're taking and the marketing channels delivering your return on investment.

This plugin allows you to link up your WordPress site with Traki Analytics.

Visit [Traki](https://app.traki.co.uk/signup "Traki") and get your account setup.

== Installation ==

1. Create an account with [Traki](https://app.traki.co.uk/signup "Traki")
2. Get your unique App ID, it'll look something similar to this "0mS537l4rEhJnYI1o1A".
3. Install the plugin by uploading it to the `/wp-content/plugins/` directory
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Go to Settings -> Traki Analytics, enter your App ID and update.
6. That's it!
